# Требования к будущему программному продукту

**Система точек продаж (POS) для операторов киосков с хот-догами**

## Описание задачи

Местному торговцу киосками с хот-догами требуется разработать современную систему точек продаж (POS), предназначенную для использования операторами киосков. Основная цель — автоматизировать обработку заказов, снизить количество ручной работы, улучшить управление запасами и обеспечить взаимодействие с клиентами через социальные сети.

Система должна быть пригодна к использованию в уличных условиях и оставаться актуальной в течение ближайших трёх лет без необходимости полной замены.

## Целевая аудитория / пользователи

- около **50 операторов киосков с хот-догами**
- **мобильные сотрудники по управлению запасами**
- **бухгалтерия**
- **тысячи клиентов**, взаимодействующих через социальные сети

## Основные требования

### Удобство использования и мобильность

- Приложение должно быть **лёгким по размеру** и работать на компактных устройствах (смартфоны, планшеты).
- Ноутбуки не рассматриваются — слишком громоздкие для работы при приготовлении хот-догов на улице.

### Функциональные требования

1. **Оформление продаж** с фиксированием времени и местоположения киоска.
2. **Поддержка скидок** — процентных, фиксированных, по акциям или промокодам.
3. **Отслеживание продаж** по времени, сменам, дням недели и отдельным точкам.
4. **Отправка обновлений по запасам** мобильным сотрудникам:

   - актуальный статус остатков,
   - задания на пополнение,
   - отметка о выполнении доставки.

5. **Интеграция с социальными сетями**:

   - информирование клиентов о ближайших открытых киосках,
   - публикация уведомлений об акциях и скидках.

6. **Экспорт данных в бухгалтерию** в формате, который может быть импортирован сторонними инструментами (например, CSV/Excel).

### Дополнительный контекст

- Текущие способы ведения учёта продаж слишком трудоёмкие и включают много ручной работы.
- **Сроки разработки критичны** — система нужна оперативно.
- Требуется решение, которое **прослужит минимум 3 года** без необходимости полной замены.
- **Бюджетных ограничений нет**, приоритет — качество, а не экономия.

## Цель разработки

Создать надёжную, удобную и расширяемую POS-систему, позволяющую автоматизировать продажи, улучшить взаимодействие с клиентами и повысить прозрачность управления запасами во всей сети киосков.
